====== 使用方法

1) IPSファイルをBN6/EXE6のROMと同じフォルダに配置してください。

2) IPSファイルの名前がパッチを適用したいROMファイルと同じ名前になっていることを確認してください。
   例： "BN6 Falzar.IPS" & "BN6 Falzar.GBA" 

3) VBA-Linkではゲームを起動したときに自動で一時的にパッチを適用することができます。

   下記の設定から自動でパッチを適用するかどうかを切り替えられます：
   'Options' -> 'Emulator' -> 'Automatic IPS patching'
   
   'Automatic IPS patching'が有効の場合、パッチが自動で適用されます。
   
   この設定でパッチを適用する場合、元のROMファイルは変更されません。

====== このパッチの特徴
対戦モード「トリプルバトル」はどちらかが69連勝するか、通信エラーが発生するまで連戦できるようになります。

対戦モード「ランダムバトル」も同様に、同じフォルダでの連戦が可能になります。

パッチが適用されていない相手と通信する場合は、パッチを適用していてもパッチ適用前と同様に振る舞います。このパッチで同期ズレは発生しません。
